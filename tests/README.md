# Estrutura de Testes

Este diretório contém os testes automatizados do projeto, organizados em:

- `unit/` — Testes unitários das funções internas do backend (src/app.py)
- `api/` — Testes de integração dos endpoints da API (src/app.py)

Execute todos os testes com:

    pytest tests/

Consulte cada subpasta para detalhes e exemplos.
